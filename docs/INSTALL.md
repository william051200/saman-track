# Installing the app

saman-track is a PWA, so it installs straight from the browser — no app store needed.
Open the deployed URL (an HTTPS site, e.g. your Vercel deployment), then:

- **Android (Chrome)** — tap the **Install app** prompt, or menu **⋮ → Install app /
  Add to Home screen**.
- **iOS (Safari)** — tap **Share** → **Add to Home Screen**.
- **Desktop (Chrome/Edge)** — click the **install icon** in the address bar, or
  menu → **Install saman-track…**.

Once installed it launches full-screen like a native app and keeps working offline.
